Mengapa membuat project website ini?

Website ini disusun untuk memenuhi tugas Mata Kuliah Kecerdasan Digital Tahun 2021.

Bagaimana proses pembagian tugas kelompok ini?

Kami melakukan analisa sesuai dengan arahan tugas, dan melakukan pembagian untuk setiap section dalam website yang kemudian digabungkan menjadi satu.

Elemen apa saja yang digunakan dalam project ini?

Kami sepenuhnya menggunakan HTML dan CSS dalam pembuatan website ini. Kami memilih menggunakan font monstrat dan menggunakan palete warna hijau dan hitam dengan desain simple. Kami juga menambahkan animasi ghost floating sebagai syarat adanya animasi yang dicantumkan dalam project ini.

Penjelasan lebih lengkap mengenai setiap section

Floating Ghost hantu tersebut akan memberikan interaksi sederhana ketika kursor diarahkan pada ghost seperti tersenyum. Menu Inside untuk variasai menu kami menggunakan Id sebagai pemanggil menu jadi tidak perlu membuat page lain, sehingga desain lebih minimalis. Accoudion Features untuk membuat informasi artikel lebih simple dalam sekali akses. Responsive Page walau layar disetting dalam berbagai ukuran infromasi mengenai about us, team dan footer tidak akan terpengaruh dan tetep stabil terbaca.
